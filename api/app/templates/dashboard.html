Getting started

Three short steps. Only the first one is required to start chatting.

1
Upload your knowledge base
Drag-and-drop your FAQ, product docs, policies (.pdf/.txt/.md). The agent will answer questions using these.
2
Put the chat widget on your website
Copy a one-line script tag. A floating chat button will appear on your site.
3
Connect integrations & CRM (optional)
Lets the agent do things — read a customer's plan, change tariff, etc. — instead of just answering. Skip if you only need Q&A.
4
Enable proactive messages (optional)
Agent reaches out first when a user's activity drops. Requires CRM step.
