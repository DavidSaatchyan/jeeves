What are agent tools?

Tools let Jeeves do things, not just answer questions. Add a Lookup to fetch live data from your systems (order status, account info, subscription details). Add an Action to let Jeeves make changes (upgrade plan, cancel order, create ticket).

Each tool is a simple HTTP call. Jeeves decides when to use it based on your description — no code needed.
