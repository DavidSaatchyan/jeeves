Authorization: Bearer sk_your_api_key_here

Content-Type: application/json

Send a message

POST to /chat with the following body:

{
  "user_id": "customer_123",
  "message": "What is my order status?"
}

Response:

{
  "response": "Your order #4567 is being prepared...",
  "action_called": null,
  "latency_ms": 1240
}

Example: cURL

curl -X POST https://your-domain.com/chat \
  -H "Authorization: Bearer sk_your_key" \
  -H "Content-Type: application/json" \
  -d '{"user_id":"customer_123","message":"Hello!"}'

Example: JavaScript

const res = await fetch("https://your-domain.com/chat", {
  method: "POST",
  headers: {
    "Authorization": "Bearer sk_your_key",
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    user_id: "customer_123",
    message: "What is my order status?",
  }),
});
const data = await res.json();
console.log(data.response);

Example: Python

import requests

resp = requests.post(
    "https://your-domain.com/chat",
    headers={"Authorization": "Bearer sk_your_key"},
    json={"user_id": "customer_123", "message": "What is my order status?"},
)
print(resp.json()["response"])
