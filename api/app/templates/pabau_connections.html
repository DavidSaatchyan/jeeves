How to get your API credentials

1. Log in to your Pabau account as an admin.

2. Go to Settings → Integrations → API Keys.

3. Generate a new API key with scopes: patients.read, patients.write, appointments.read, appointments.write.

4. Copy the API key and your Company ID (found in Settings → Account).

5. Webhook URL: {{ request.url.scheme }}://{{ request.url.host }}/integrations/webhooks/pabau/{{ tenant_id }}
